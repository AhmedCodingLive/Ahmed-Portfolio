About

I'm a passionate YouTuber and web developer with a background in content creation, website development, and basic programming. I started my YouTube journey in 2018, creating engaging and informative videos that helped me build a dedicated audience and develop strong communication skills. In 2019, I began learning web development, focusing on HTML, CSS, and JavaScript. Over time, I built responsive websites and developed a solid understanding of front-end development principles. In 2023, I explored game development by creating a 2D ball-and-door puzzle game using GDevelop 5, which introduced me to event-based logic and interactive design. In 2024, I started learning the basics of C++ programming, gaining an introduction to core concepts such as variables, control structures, functions, and simple object-oriented programming.
